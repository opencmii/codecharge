Components: Folder that contains the re-usable CodeChargeStudio Design Templates.
Provided here as contribution to the clf effort, to facilitate its integration with Rapid
Application Developpment (RAD) made with CodeChargeStudio 5.X

Usage:
Install CodechargeStudio5
Copy Components directly in C:\Program Files\CodeChargeStudio5, to update the list of available Design Templates.


Projects: Folder that contains zero or more CodeChargeStudio based demo application (in progress), making use of the design template above.

Usage
Install CodechargeStudio5
Copy Projects in C:\Program Files\CodeChargeStudio5, to update the list of local projects. 
This folder is only required if you want to re-generate or modify the demo application provided.
